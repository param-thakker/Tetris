***Tetris***


### Mac

#### Install Homebrew
```
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
# Check for success
brew doctor


brew install sdl2


## Build & Run Game
```
git clone https://github.com/param-thakker/Tetris.git
cd tetris
make
./tetris
```
